========================================
GitHub Actions 自动构建说明
========================================

【工作流文件位置】
GitHub 仓库：.github/workflows/build-windows-installer.yml
本地备份：build-scripts/build-windows-installer.yml

【使用方法】
1. 打开 GitHub 仓库页面
2. 点击 "Actions" 标签
3. 选择 "Build Windows Installer" 工作流
4. 点击 "Run workflow" 按钮
5. 选择分支（main 或 master）
6. 点击 "Run workflow" 确认

【构建步骤】
工作流会自动执行：
1. Checkout 代码
2. Setup Node.js 18
3. Install dependencies
4. Build server
5. Compile electron
6. Build frontend
7. Rebuild native modules（新增）
8. Build Windows installer
9. Copy installer to build-scripts（新增）
10. Upload installer as artifact

【下载安装包】
构建完成后：
1. 在 Actions 页面找到完成的构建
2. 点击构建记录
3. 在 "Artifacts" 部分下载 windows-installer
4. 解压后找到：仓库管理-1.0.0-Setup.exe

【构建时间】
- 首次构建：10-15 分钟
- 后续构建：5-10 分钟（依赖已缓存）

【优势】
- 无需本地 Node.js
- 自动构建，一键触发
- 自动 rebuild native 模块
- 自动复制到 build-scripts
- 自动上传 artifact
